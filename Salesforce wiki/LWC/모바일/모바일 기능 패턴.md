# 모바일 기능 패턴

> Salesforce 모바일 앱 전용 API — 기기 GPS와 바코드 스캐너 사용. 브라우저에서도 동작하는 fallback과 함께 구현해야 함.

---

## isAvailable() 가드 — 항상 먼저

```javascript
connectedCallback() {
    this.myScanner = getBarcodeScanner();
    if (this.myScanner?.isAvailable()) {
        this.scanButtonEnabled = true; // 모바일 앱에서만 활성화
    }
    // isAvailable() false면 버튼 숨김 — 에러 없음
}
```

> `isAvailable()` — Salesforce 모바일 앱 내에서만 `true`. 브라우저에서는 항상 `false`.

## 바코드 스캐너 패턴

```javascript
import { LightningElement } from 'lwc';
import { NavigationMixin } from 'lightning/navigation';
import { ShowToastEvent } from 'lightning/platformShowToastEvent';
import { getBarcodeScanner } from 'lightning/mobileCapabilities';

export default class BarcodeScanner extends NavigationMixin(LightningElement) {
    myScanner;
    scanButtonEnabled = false;

    connectedCallback() {
        this.myScanner = getBarcodeScanner();
        if (this.myScanner?.isAvailable()) {
            this.scanButtonEnabled = true;
        }
    }

    async handleBeginScanClick() {
        if (!this.myScanner?.isAvailable()) return;

        const scanningOptions = {
            barcodeTypes: [this.myScanner.barcodeTypes.QR],
            instructionText: 'Scan a QR Code',
            successText: 'Scanning complete.'
        };

        try {
            const result = await this.myScanner.beginCapture(scanningOptions);
            const scannedId = result.value;

            // QR 코드 값으로 레코드 페이지 이동
            this[NavigationMixin.Navigate]({
                type: 'standard__recordPage',
                attributes: { recordId: scannedId, actionName: 'view' }
            });
        } catch (error) {
            if (error.code === 'userDismissedScanner') {
                this.dispatchEvent(new ShowToastEvent({
                    title: 'Scanning Canceled',
                    message: 'Scanning session canceled.',
                    mode: 'sticky'
                }));
            } else {
                this.dispatchEvent(new ShowToastEvent({
                    title: 'Barcode Scanner Error',
                    message: 'Error: ' + error.message,
                    variant: 'error',
                    mode: 'sticky'
                }));
            }
        } finally {
            this.myScanner.endCapture(); // 항상 세션 종료
        }
    }
}
```

### 에러 코드

| 코드 | 상황 |
|---|---|
| `userDismissedScanner` | 사용자가 취소 |
| 그 외 | 하드웨어/권한 오류 |

## 위치 서비스 패턴 (모바일 + 브라우저 fallback)

```javascript
import { getLocationService } from 'lightning/mobileCapabilities';
import { getRecord, getFieldValue } from 'lightning/uiRecordApi';

const LATITUDE_FIELD  = 'Property__c.Location__Latitude__s';
const LONGITUDE_FIELD = 'Property__c.Location__Longitude__s';

export default class PropertyLocation extends LightningElement {
    error;
    deviceLocationService;
    location;
    @api recordId;

    @wire(getRecord, { recordId: '$recordId', fields: [LATITUDE_FIELD, LONGITUDE_FIELD] })
    wiredProperty({ data, error }) {
        if (data)  { this.property = data; this.calculateDistance(); }
        if (error) { this.error = error; }
    }

    async connectedCallback() {
        this.deviceLocationService = getLocationService();

        if (this.deviceLocationService.isAvailable()) {
            // 모바일 앱 — 높은 정확도
            await this.calculateLocationFromMobileDevice();
        } else if (navigator.geolocation) {
            // 브라우저 — Geolocation API fallback
            this.calculateLocationFromBrowser();
        } else {
            this.error = { message: 'No location services available' };
        }
    }

    async calculateLocationFromMobileDevice() {
        try {
            this.location = await this.deviceLocationService.getCurrentPosition({
                enableHighAccuracy: true
            });
            this.calculateDistance();
        } catch (error) {
            this.error = error;
        }
    }

    calculateLocationFromBrowser() {
        navigator.geolocation.getCurrentPosition(
            (result) => { this.location = result; this.calculateDistance(); },
            (error)  => { this.error = error; }
        );
    }

    calculateDistance() {
        if (!this.location || !this.property) return;

        const lat1 = this.location.coords.latitude;
        const lat2 = getFieldValue(this.property, LATITUDE_FIELD);
        const lon1 = this.location.coords.longitude;
        const lon2 = getFieldValue(this.property, LONGITUDE_FIELD);

        // Haversine formula
        const deg2rad = (deg) => (deg * Math.PI) / 180.0;
        const R  = 6371;
        const dLat = deg2rad(lat2 - lat1);
        const dLon = deg2rad(lon2 - lon1);
        const a = Math.sin(dLat/2)**2 +
                  Math.cos(deg2rad(lat1)) * Math.cos(deg2rad(lat2)) * Math.sin(dLon/2)**2;
        const d = R * 2 * Math.atan2(Math.sqrt(a), Math.sqrt(1-a));
        this.distance = d / 1.609344; // km → miles
    }
}
```

## 모바일 vs 브라우저 결정 흐름

```
connectedCallback()
  ├── getLocationService().isAvailable() → true  → 모바일 API (async/await)
  ├── navigator.geolocation  → true              → 브라우저 Geolocation API
  └── 둘 다 없음                                 → error 상태 표시
```
